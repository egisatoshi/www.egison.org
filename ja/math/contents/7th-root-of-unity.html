1の7乗根

Apr 7 2016

1の7乗根の冪根表示

1の7乗根について、

`cos((2pi)/7) = 1/6 (-1 + root 3 ((7 + 21 sqrt 3 i) / 2) + root 3 ((7 - 21 sqrt 3 i) / 2))`

であることが知られています。
以下のコードはこの値を実際に計算して求めています。

`x^6 + x^5 + x^4 + x^3 + x^2 + x + 1 = 0`

のガロア群が`ZZ//7ZZ-{0}`における乗法についての循環群と同じ構造であることを利用した解法で計算しています。

方程式を解くもう1つの道筋

上記の方法は、三次方程式を解いたあとに二次方程式を解くことにより、1の7乗根を求めました。
この順番を逆にして、二次方程式を解いたあとに三次方程式を解くことにより、1の7乗根を求めることも可能です。
