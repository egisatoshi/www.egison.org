Egisonで微分計算(高校編)


問題

Egisonを使って以下のように微分計算をすることができます。

`(d x^2) / (d x) = 2 x`
`(d x^3) / (d x) = 3 x^2`
`(d cos(x)) / (d x) = - sin(x)`
`(d x sin(x)) / (d x) = sin(x) + x cos(x)`
`(d e^(3 x)) / (d x) = 3 e^(3 x)`
`(d log(x)) / (d x) = 1 / x`

回答方法

空欄[ 1 ]を埋めるEgisonプログラムを、問題中のサンプルと同じコーディングスタイルで記述し、提出してください。
