1の原始5乗根の計算


問題

1の5乗根とは、5乗したら1になる数のことをいいます。
つまり、方程式`z^5 - 1 = 0`の解のことをいいます。

特に1の原始5乗根とは、5乗して始めて1になる数のことをいいます。
これは、方程式`z^5 - 1 = (z - 1) (z^4 + z^3 + z^2 + z + 1) = 0`の1以外の解になります。
つまり、方程式`z^4 + z^3 + z^2 + z + 1 = 0`の解は1の原始5乗根です。
本問題ではこの方程式を解き、1の原始5乗根の値を求めてみましょう。

方程式`z^4 + z^3 + z^2 + z + 1 = 0`について、以下のような解法が知られています。

まず、

`z^4 + z^3 + z^2 + z + 1 = 0`

の両辺を`z^2`で割ります。(`z != 0`は明らか)
その結果得られた方程式

`z^2 + z + 1 + 1/z + 1/z^2 = 0`

を、

`(z + 1/z)^2 + (z + 1/z) - 1 = 0`

と変形します。
この方程式について、

`t = z + 1/z`

と置換すれば、2次方程式を2回解くことにより、1の5乗根を求めることができます。

さて、実際にEgisonを用いてこの方法により、1の原始5乗根を求めてみましょう。
下記1行目、4行目は、それぞれ`t`、`z`の値を求め、それぞれを変数t、zに束縛するEgisonプログラムとなっています。
それらに続く2行目、5行目は、`t`、`z`に束縛された値をコメントアウトしたものとなっています。
7行目、8行目では、求めたzの値を5乗すると1になることを確かめています。

上記のプログラムに現れているfst関数はタプルを引数にとり、そのタプルの1つ目の要素を返す関数です。
ここでは、2次方程式の2つの解の片方を取得するために使われています。

問題1

空欄[ 1 ]を埋めるEgisonプログラムを、以下の4択から選んでください。
q-f'関数については、CodeIQ掲載記事を参考にしてください。

  • 1: (q-f' 1 0 1)
  • 2: (q-f' 1 0 -1)
  • 3: (q-f' 1 1 -1)(正解)
  • 4: (q-f' 1 -1 1)

問題2

空欄[ 2 ]を埋めるEgisonプログラムを、以下の4択から選んでください。

  • 1: (q-f' 1 t -1)
  • 2: (q-f' 1 t 1)
  • 3: (q-f' 1 (* -1 t) -1)
  • 4: (q-f' 1 (* -1 t) 1)(正解)

問題3

結果として得られた1の原始5乗根の値として正しいものを選んでください。

  • 1: `(-1 + sqrt 5 + sqrt (-10 - 2 sqrt 5))/4`(正解)
  • 2: `(-1 + sqrt 5 - sqrt (10 + 2 sqrt 5))/4`
  • 3: `(-1 + sqrt 5)/2`
  • 4: `(-1 + sqrt (-5))/2`
