Leibniz formula

Let's consider a sawtooth wave function.

`f(x) = x`       `(-pi < x < pi)`
`f(x + 2 n pi) = f(x)`

And get the Fourier coefficients of this function.

`f(x) = a_0 / 2 + sum_(k=0) a_k cos(k x) + sum_(k=0) b_k sin(k x)`

In the following program, we caclcluate these expressions to get the Fourier coefficients.

`a_0 = 1 / pi int_(-pi)^pi f(x) dx = 0`

`a_k = 1 / pi int_(-pi)^pi f(x) cos(k x) dx = 0`

`b_k = 1 / pi int_(-pi)^pi f(x) sin(k x) dx = (2 (-1)^k) / k`

Therefore,

`f(x) = sum_(k=0) ((2 (-1)^k sin(k x)) / k)`

If we substitute `x` with `pi / 2`, we get Leibniz formula as follow.

`pi / 2 = -2 + 1 - 2 / 3 + 1 / 2 - 2 / 5 + 1 / 3 - 2 / 7 + 1 / 4 - 2 / 9 + 1 / 5 - ...`

`pi / 4 = -1 + 1 / 2 - 1 / 3 + 1 / 4 - 1 / 5 + 1 / 6 - 1 / 7 + 1 / 8 - 1 / 9 + 1 / 10 - ...`
